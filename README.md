# node-es6-sample
> NodeJS Sample with ES6 support.


This uses

* ES6 ( Using babel-register )
* NodeJS ( version >= 6 )
* Express
* Mongoose
* Body-parser
* Nodemon

### Get Started
Open up your terminal
```
git clone https://github.com/sivaram143/node-es6-sample.git
cd node-es6-sample
npm install
```
Start your mongodb server instance
```
mongod
```

Start dev server
```
npm run dev
```
